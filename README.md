# VetClinic - Base inicial

Proyecto base para el Sistema de Gestión para Clínica Veterinaria.

¿Qué incluye esta base?
- Estructura de proyecto Spring Boot
- Entidades base User/Role
- Repositorios JPA
- Controlador de registro mínimo
- Configuración de seguridad básica
- Mappers con MapStruct
- Manejo básico de excepciones
- `application.yml` con placeholders para Postgres

Cómo ejecutar (con Maven):

1. Ajusta variables de entorno para la BD o usa la configuración por defecto.
2. Ejecuta: mvn spring-boot:run
